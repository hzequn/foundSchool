# foundSchool
校园失物招领平台

author: 13414851033@163.com   

# 系统介绍 

> 校园失物招领平台是构建学生为主体的一个方便学生快速找到失主/拾主的系统。主体功能是发布失物招领信息以及系统自动匹配可能的拾｜失物品。

## 核心技术

- 前端
  + Html+CSS布局
  + Js+smarty模板引擎框架实现数据渲染
  
- 后台
  + Php后台语言
  + MySql数据库
  + 为学生提供招领信息的符合度匹配
